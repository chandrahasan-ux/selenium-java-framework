# Selenium Java Automation Framework

A QA automation framework built with Selenium WebDriver, TestNG, and Maven.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## What's inside
- Automated test scripts for web application testing
- Page Object Model (POM) design pattern
- Reusable utility methods
